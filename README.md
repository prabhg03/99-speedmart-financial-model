# Financial Modeling & Valuation Project – 99 SpeedMart Retail Holdings Berhad

## Overview

This project presents a financial analysis and valuation model for 99 SpeedMart Retail Holdings Berhad. The objective of the model is to analyze historical financial performance and estimate the intrinsic value of the company using fundamental valuation techniques.

The model was built in Microsoft Excel and includes financial statement analysis, revenue forecasting, discounted cash flow (DCF) valuation, and comparable company analysis.

## Project Structure

The Excel model is organized into the following worksheets:

**Data**

* Contains historical financial statement data extracted from company reports.
* Includes revenue, cost of goods sold, gross profit, net profit, assets, and liabilities.

**FSA (Financial Statement Analysis)**

* Analyzes key financial metrics and trends over the historical period.
* Calculates revenue growth and profitability margins to understand business performance.

**DCF Model**

* Forecasts future financial performance based on historical growth trends.
* Estimates future free cash flows and calculates company valuation using the Discounted Cash Flow (DCF) method.
* Includes key assumptions such as revenue growth, discount rate (WACC), and terminal growth rate.

**Comps (Comparable Company Analysis)**

* Benchmarks the company against industry peers using valuation multiples.
* Provides an additional perspective for assessing relative valuation.

## Methodology

The valuation model follows a standard corporate finance approach:

1. Historical financial data was collected from company reports.
2. Financial statement analysis was performed to identify growth trends and margins.
3. Revenue and profitability were forecasted using historical averages and reasonable assumptions.
4. Free cash flows were projected and discounted using a weighted average cost of capital (WACC).
5. The intrinsic value estimate was compared with market valuation using comparable company multiples.

## Tools Used

* Microsoft Excel
* Financial statement analysis
* Discounted Cash Flow (DCF) valuation
* Comparable company analysis
